# Pander

This is an app for prototyping games where the player panders to a simulated crowd.

## Licensing

Code and other files in this repository are licensed under the MIT open source license.

The fonts are hosted from decentapps.net rather than included in this repo. Their licensing is separate, but will be something compatible with open source, e.g., OFS SFIL.

## Community

Pander is a Decent App, which is a kind of web app that sends no user data to a server, using all local capabilities. You can find users of Pander and other Decent Apps on the [Decent Apps Discord server](https://discord.gg/kkp3x4X2Vb).
